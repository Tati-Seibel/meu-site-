Como Publiquei Meu Site no GitHub Pages

Criei meu repositório no GitHub Entrei no GitHub e criei um repositório novo para o meu site. Enviei os arquivos do site Fiz o upload das pastas e arquivos do site diretamente para o repositório. Publiquei com o GitHub Pages Acessei Settings → Pages e ativei o GitHub Pages. O GitHub gerou um link para meu site, que ficou assim: https://tati-seibel.github.io/meu-site-/. Visualizei offline Baixei pelo aplicativo com o Edge. Assim pude abrir os arquivos localmente e ver meu site funcionando offline, com opção de navegar entre Home e SObre mim.
